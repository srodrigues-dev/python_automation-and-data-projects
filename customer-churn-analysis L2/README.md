# Customer Churn Analysis 📊

## 📋 Project Overview
A data analysis project focused on understanding customer rotativity (churn) for a subscription company. The goal is to analyze the database, identify the core reasons why customers cancel their services, and propose automated or strategic data-driven solutions to increase user retention.

## 🛠️ Technologies & Tools
* **Language:** Python
* **Libraries:** Pandas, Plotly (Data Visualization)
* **Environment:** Jupyter Notebook (`analysis.ipynb`)
