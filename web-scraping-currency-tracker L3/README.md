# Web Scraping Currency Tracker 🌐

## 📋 Project Overview
A web scraping automation project that navigates the web to gather live financial data, such as currency quotes. Once the data is retrieved, the script automatically updates a local customer database (`customers.csv`), mapping current values directly to user profiles.

## 🛠️ Technologies & Tools
* **Language:** Python
* **Libraries:** Selenium / BeautifulSoup, Pandas, PyAutoGUI
