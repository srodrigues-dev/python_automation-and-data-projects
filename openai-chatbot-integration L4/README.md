# OpenAI Chatbot Integration 🤖

## 📋 Project Overview
A script showcasing the integration of generative Artificial Intelligence within a local workflow. It establishes a secure connection with the OpenAI API to simulate automated client responses, acting as an auxiliary backend engine for customer support bots.

## 🛠️ Technologies & Tools
* **Language:** Python
* **AI Engine:** OpenAI API
* **Libraries:** OpenAI, Python-dotenv (Environment Variables)
