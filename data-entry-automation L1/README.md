# Data Entry Automation Script 📥

## 📋 Project Overview
An automation script developed to eliminate repetitive tasks by automatically filling out an enterprise registration system. It reads a local product database and simulates human inputs (mouse clicks and keystrokes) to upload products into the company's platform efficiently.

## 🛠️ Technologies & Tools
* **Language:** Python
* **Libraries:** PyAutoGUI, Time, Pandas
* **Environment:** VS Code
